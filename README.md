# 📊 Predictive Analytics Using Historical Data

## 📌 Overview

This project focuses on analyzing historical sales data to identify trends and predict future revenue using machine learning techniques.

## 🎯 Objective

To build a predictive model that can forecast future sales based on past data and identify patterns such as trends and seasonality.

## 📊 Dataset

The dataset contains:

* Date
* Product
* Category
* Price
* Quantity
* Revenue

## ⚙️ Steps Performed

* Data cleaning and preprocessing
* Conversion of daily data into monthly data
* Exploratory Data Analysis (EDA)
* Trend visualization using graphs
* Model building using Linear Regression
* Future prediction of revenue

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib
* Scikit-learn

## 📈 Results

* Identified trends in revenue over time
* Observed fluctuations indicating seasonal patterns
* Successfully predicted future revenue values

## ✅ Conclusion

The project demonstrates how historical data can be used to make data-driven predictions and support decision-making.
